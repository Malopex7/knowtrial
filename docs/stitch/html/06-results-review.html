Which protocol is primarily used for assigning dynamic IP addresses in a VCP-standard virtualized network environment?

C

DHCP (Dynamic Host Configuration Protocol)

Your Answer
A

DNS (Domain Name System)

info Explanation

DHCP is the standard protocol for dynamic IP allocation. In a VMware environment, the NSX-T manager or traditional DHCP relays are used to ensure that virtual machines receive unique IP addresses upon power-on within their respective subnets.

Question 13

What is the default MTU (Maximum Transmission Unit) size required for VXLAN encapsulation to function correctly without fragmentation?

B

1500 bytes

Your Incorrect Choice
D

1600 bytes

Correct Answer
auto_stories Learning Rationale

VXLAN adds a 50-byte header to the original Ethernet frame. To accommodate this overhead on physical network links, the MTU must be increased. While 1550 is the minimum, 1600 is the recommended standard for enterprise VCP deployments to ensure compatibility with various overlay technologies.
